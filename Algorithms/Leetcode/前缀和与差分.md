## 差分

```C
// 原数组
int a[5] = {1,2,3,4,7};

// 原数组的差分数组
int b[5] = {1,1,1,1,3};

// 规律
b[i] = a[i]; // i=0
b[i] = a[i] - a[i-1]; // i>0

// 多次对a[L,R]之间数加上一个数val（val为整数），求最终数组a的结果
b[L] += val; // R=0
b[R+1] -= val; // R+1 < N（N为数组a长度）
```
